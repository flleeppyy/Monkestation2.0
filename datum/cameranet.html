proc
get_available_camera_by_tag_list
(/list/networks_available, /list/z_levels_available)

get_available_camera_by_tag_list

Builds a list of all available cameras that can be seen to networks_available and in z_levels_available. Entries are stored in c_tag[camera.can_use() ? null : " (Deactivated)"] => camera format Args: networks_available - List of networks that we use to see which cameras are visible to it. z_levels_available - List of z levels to filter camera by. If empty, all z levels are considered valid.

proc
major_chunk_change
(/atom/center_or_camera, choice, update_delay_buffer)

Never access this proc directly!!!! This will update the chunk and all the surrounding chunks. It will also add the atom to the cameras list if you set the choice to ADD_CAMERA. Setting the choice to REMOVE_CAMERA will remove the camera from the chunks. If you want to update the chunks around an object, without adding/removing a camera, use IGNORE_CAMERA. update_delay_buffer is passed all the way to queue_update() from portable camera updates on movement to change the time between static updates.
